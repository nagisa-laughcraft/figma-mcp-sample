# Figma MCP Sample

このリポジトリは、Figma MCPのサンプルプロジェクトです。

## 概要

Figmaとの連携機能を実装したサンプルプロジェクトです。

## 機能

- Figmaデザインの取得
- コンポーネントの自動生成
- デザインの同期

## セットアップ

1. リポジトリのクローン
```bash
git clone https://github.com/nagisa-laughcraft/figma-mcp-sample.git
cd figma-mcp-sample
```

2. 依存関係のインストール
```bash
npm install
```

3. 環境変数の設定
```bash
cp .env.example .env
# .envファイルを編集して必要な環境変数を設定
```
