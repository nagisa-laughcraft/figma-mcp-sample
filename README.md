# Figma MCP Sample

Next.jsを使用したFigmaプラグインのサンプルプロジェクトです。

## 開発環境のセットアップ

```bash
npm install
```

## 開発サーバーの起動

```bash
npm run dev
```

## テストの実行

```bash
# ユニットテスト
npm run test

# E2Eテスト
npm run test:e2e
```

## 品質確認

```bash
# 型チェック
npm run typecheck

# リンター
npm run lint
```

## ビルド

```bash
npm run build
```
